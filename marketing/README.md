# Abeam Marketing Plugin

Marketing operations toolkit for Abeam — a pilot-first platform for general aviation.

## Skills

### brand-voice
Abeam's brand voice and tone guidelines. Loads automatically whenever Claude generates any marketing copy for Abeam. Contains the do/don't table, aviation terminology rules, audience tone adjustments, and competitor mention guidelines. Reference files include the full tagline library, audience-specific messaging, and boilerplate copy at various lengths.

## Planned Components

### Future Skills
- **tagline-library** — Full tagline system by category (currently a reference file within brand-voice)
- **audience-profiles** — Detailed buyer personas for student pilots, instructors, and school operators
- **competitive-context** — Competitor positioning, feature comparisons, and market gaps
- **platform-specs** — Character limits, image sizes, and best practices per social platform
- **experiment-memory** — How to read/write the content experiment log
- **product-knowledge** — Current product capabilities, recent releases, feature inventory

- `/abeam:performance-review` — Analyze experiment log
